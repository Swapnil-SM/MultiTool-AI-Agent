This project is a personal AI assistant that helps automate daily productivity tasks.

The goal of this project is to combine a language model with different real-world tools like Gmail, Google Calendar, task management, and expense tracking into one single system.

## What This Agent Can Do

• Answer questions using web search  
• Create and manage Google Calendar events  
• Read emails, summarize them, and send replies  
• Manage tasks and to-do lists  
• Take quick notes  
• Track expenses and manage budgeting  

## How It Works

The system connects a language model with different external APIs like:
- Google Calendar API
- Gmail API
- Task management tools
- Expense tracking tools
- Web search APIs
